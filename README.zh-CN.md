# BeforeInstall（中文说明）

English version: [README.md](README.md)

BeforeInstall 是一个 macOS 下的安装前与运行行为风险分析工具。

## 当前状态

项目仍在持续开发中，整体稳定性还不高。

## 主要功能

- 对大部分文件做基础的分析
- 全盘扫描（快速/深度）与风险筛选
- 受限环境下的运行行为观察（动态分析）
- 风险摘要与报告导出
- 在设置中可选启用随机森林模型推理（WIP）

## 安装方式

### 方式一：安装发布包

1. 下载并打开发布包。
2. 将 `BeforeInstall.app` 拖到 `Applications`。
3. 若被 macOS 阻止，请前往 `系统设置 > 隐私与安全性`，点击 `仍要打开`。

### 方式二：源码构建

环境要求：

- macOS 13+
- 带 Swift 6 工具链的 Xcode

构建：

```bash
open BeforeInstall.xcodeproj
```

然后在 Xcode 中运行 `BeforeInstall` target。

## 首次使用权限说明（重要）

为了保证全盘扫描效果，请手动授予权限：

- `系统设置 > 隐私与安全性 > 完全磁盘访问权限`
- 为 `BeforeInstall` 打开开关

未授予时，部分路径可能无法扫描或仅能部分扫描。


## 免责声明

- 预测和风险评分结果**不保证准确**。
