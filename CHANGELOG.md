# BeforeInstall Changelog

## 1.0.0beta2

Date: 2026-03-29

- Reworked the app chrome around Apple's Liquid Glass system styles instead of custom gradients, making the main window and settings window closer to the native macOS translucent look.
- Replaced custom top-level navigation pills and key option selectors with native palette-based controls where available, while keeping existing workflows unchanged.
- Unified glass buttons and panels to improve text readability and remove the heavy blue tint from the previous beta build.
